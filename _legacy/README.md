# Веб-приложение для заказа еды

**Курсовая работа**
*Тема: Разработка веб-приложения для заказа еды*
Дисциплина: Интернет-технологии

## Описание

Интернет-магазин доставки еды: каталог блюд по категориям, поиск и фильтры,
корзина с подсчётом суммы и промокодами, оформление заказа с валидацией формы
и страница подтверждения.

## Как это устроено

Приложение полностью клиентское: вся логика и данные (каталог из 23 блюд)
находятся в браузере (`frontend/js/api.js`), корзина хранится в `localStorage`.
Поэтому сайт работает и при простом открытии файла, и на GitHub Pages —
без какого-либо сервера.

## Технологии

- HTML5, CSS3, нативный JavaScript (ES6+)
- localStorage для корзины и заказов
